# Music en Vivo

## Purpose
A simple, exciting website that serves as a hub for music lovers to find concerts from the artists they love. Enter a muscician's name to find events for that musician.

## Built With...
* HTML
* CSS
* Javascript
* Ticketmaster API
* Spotify API
